## 项目介绍

课程源码见：[build-zhihu-with-springboot-and-tdd](https://github.com/qianhuihuiji/build-zhihu-with-springboot-and-tdd)

本项目的初衷是打造一门兼具实用性和易学性的测试课程。本项目将以 TDD 为切入口，讲解 Web 应用常见功能的测试，如：

- 数据库测试
- 测试表单验证
- 测试文件上传
- 测试邮件发送
- 测试远程 API 调用
- 测试「异常」抛出
- 模拟登录用户
- contract 测试

涉及了 Web 应用常见的测试层级，包括：

- 单元测试
- 集成测试
