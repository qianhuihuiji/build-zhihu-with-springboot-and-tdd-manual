## spring-boot-starter-test

包含以下提供的库：

- JUnit 5: 单元测试Java应用程序的事实标准。
- Spring Test & Spring Boot Test: 为Spring Boot应用程序提供实用程序和集成测试支持。
- AssertJ: 一个 fluent 风格的断言库。
- Hamcrest: 一个匹配器对象库（也被称为约束或谓词）。
- Mockito: 一个 Java mocking 框架.
- JSONassert: 一个用于JSON的断言库。
- JsonPath: 用于JSON的XPath。

> https://springdoc.cn/spring-boot/features.html#features.testing

## Testcontainer

利用 Docker 在测试时创建开箱即用的中间件实例，例如 MySql、Redis等，可以做到每个人、每次测试均使用全新的环境，最大化保证测试环境的干净。本项目还配置了`reuse`的属性，这意味着，在创建过的容器运行的情况下，`Testcontainers `将会复用该容器，而不是新创建一个。

> 注： Testcontainers 的容器复用功能有一个硬性前提：仅复用「处于运行状态」的容器，已停止的容器会被框架判定为「不可用资源」，不会尝试重启或复用，而是直接新建。
