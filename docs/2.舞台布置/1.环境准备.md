本文档不涉及环境的搭建，仅列出需要的环境依赖列表。

## JDK 17

首先，你需要搭建好 JDK 17 的环境（或者 JDK 17 更高的版本）。

## IntelliJ IDEA

代码开发的工具，社区版或者正式版均可。

## Maven

Java 包管理工具，安装好 IntelliJ IDEA 可以直接用自带的 Maven 工具。

## Docker

本项目中使用到了 Mysql、Kafka、Redis 等中间件，均使用 Docker 进行构建。且功能测试中，Docker 是强依赖，因此请务必安装并配置好
Docker 的环境。
> 注：当前笔者的 Docker 版本为 28.5.1
